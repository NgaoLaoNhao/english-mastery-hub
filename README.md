\# English Mastery Hub



Qt 6 desktop app theo dõi thử thách 25 ngày học tiếng Anh nhóm.



\## Tech Stack

\- \*\*Qt 6.11\*\* (Quick + QuickControls2 + Sql + Charts)

\- \*\*C++17\*\*, CMake 3.21+

\- \*\*SQLite\*\* (qua QSqlDatabase)



\## Features

\- Đăng nhập + bắt buộc đổi mật khẩu lần đầu

\- Daily check-in (Bookworm + Ministory)

\- Wanted Board (nhắc người chưa điểm danh)

\- Personal Area (calendar 25 ngày + 2 line chart)

\- Group Comparison (bar + line chart)

\- Top Performer Podium (huy chương 🥇🥈🥉)

\- Resource Library (link + file)

\- Reminder 21:00, Export PDF, Theme sáng/tối



\## Build

1\. Cài \*\*Qt 6.11\*\* + \*\*MinGW 64-bit\*\* + \*\*Qt Charts\*\* (qua Qt Maintenance Tool).

2\. Mở `CMakeLists.txt` bằng \*\*Qt Creator\*\*.

3\. Configure project với kit \*Desktop Qt 6.11.0 MinGW 64-bit\*.

4\. Bấm \*\*Run\*\* (Ctrl+R).
